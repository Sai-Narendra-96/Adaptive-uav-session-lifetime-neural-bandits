# UAV session lifetime

This repository contains the code and the simulation data used in our paper on
session lifetime selection in an Internet of Drones network.

The main idea is simple. After a UAV completes authentication with the ground
station, the ground station has to choose how long the session should remain
active. We study this choice using fixed timeout rules, Thompson Sampling,
LinUCB, LinearTS and NeuralTS.

## Files

- `code/run_experiment.py` runs the full experiment and creates the results and
  figures used in the paper.
- `code/make_data.py` creates the CSV files for seeds 0 to 19.
- `data/` contains one CSV file for each seed.

Each CSV file contains 30,000 simulation rounds. The files are named from
`UAV Simulation for seed 0.csv` to `UAV Simulation for seed 19.csv`.

## About the data

The data are synthetic. They are generated by the simulator and are not
measurements collected from real UAV flights.

The simulator generates UAV position, speed, remaining energy, heading and
channel state for every round. It then calculates the session lifetime used as
the reference value in each of the three channel environments.

For more details about the columns, see `data/README.md`.
